# Resume Forge ⚒

An AI-powered resume optimization tool built with React that tailors your resume to specific job descriptions, scores ATS compatibility, and generates cover letters — all in one workflow.

## Live Demo

🔗 [resume-forge-nu.vercel.app](https://resume-forge-nu.vercel.app)

## Features

- **Resume optimization** — paste your resume and a job description, and the AI rewrites your resume to match the role, adjusting tone, format, and keyword density
- **ATS scoring** — before/after ATS compatibility scores with a full breakdown by category and keyword gap analysis
- **Cover letter generation** — auto-generates a tailored cover letter based on your resume and the job description
- **Job application tracker** — save applications with status tracking (Applied, Interview, Offer, Rejected)
- **PDF export** — download your optimized resume and cover letter as formatted PDFs
- **Customizable preferences** — choose from multiple formats, visual styles, tones, and color palettes

## Tech Stack

| Layer | Tools |
|-------|-------|
| Frontend | React, JavaScript |
| AI | Claude API (Anthropic) |
| PDF Generation | jsPDF |
| Deployment | Vercel |

## How It Works

1. Click **Load Demo** to pre-fill a sample resume and job description
2. Select your preferences — format, tone, length, color palette
3. Click **⚒ Forge my resume**
4. Review results across three tabs: Optimized Resume, ATS Score, Cover Letter
5. Download as PDF or save to the job tracker
